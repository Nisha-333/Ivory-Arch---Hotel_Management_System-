# Ivory Arch — Hotel Management System

A JavaFX desktop application for hotel management built with Java 21.

## Abstract
Ivory Arch Hotel Management System is a standalone desktop application built with Java 21 and JavaFX. It implements a full MVC architecture backed by a SQLite database. The system supports two roles — Admin and Guest — with dedicated dashboards. Core features include room management, booking, checkout, billing with GST, housekeeping, analytics, and smart room recommendation. The project demonstrates all ten weeks of Java lab concepts including OOP, multithreading, file I/O, serialization, generics, collections, and JDBC.


## How to Run
Requires JDK 21 and Maven 3.6+
```bash
mvn clean javafx:run
```
